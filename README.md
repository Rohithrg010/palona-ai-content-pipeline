# 🚀 AI-Powered Marketing Content Pipeline

## 📌 Overview
This project builds an automated AI-driven marketing pipeline for NovaMind, a fictional AI startup helping creative agencies automate workflows.

The pipeline:
1. Generates blog content and personalized newsletters
2. Segments users and simulates CRM-based distribution
3. Tracks engagement metrics
4. Uses AI-style analysis to suggest improvements

---

## 🧠 Features

### 1. Content Generation
- Blog title, outline, and 400–600 word blog
- 3 persona-based newsletters:
  - Agency Owners (growth-focused)
  - Operations Managers (efficiency-focused)
  - Creative Leads (execution-focused)

### 2. CRM Simulation
- Creates mock contacts
- Segments users by persona
- Sends targeted newsletters
- Logs campaign activity

### 3. Performance Analysis
- Simulates:
  - Open rate
  - Click rate
  - Unsubscribe rate
- Identifies best-performing persona
- Suggests improvements and next content ideas

---

## ⚙️ Tech Stack
- Python
- JSON (data storage)
- OpenAI (mocked due to API quota)
- VS Code
- GitHub
